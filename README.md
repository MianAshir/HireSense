# HireSense 💼🤖

[![LinkedIn Demo](https://img.shields.io/badge/LinkedIn-Watch_Demo_Post-0A66C2?style=for-the-badge&logo=linkedin)](https://www.linkedin.com/posts/ashir-bin-asif_artificialintelligence-aiagents-n8n-ugcPost-7474576971583307776-Z2hC/?highlightedUpdateUrn=urn%3Ali%3Aactivity%3A7474718193941667840&highlightedUpdateType=SOCIAL_SHARE&origin=SOCIAL_SHARE&utm_source=share&utm_medium=member_desktop&rcm=ACoAAGNsZugB1_OKBssqW4a1NonfQviiGVhJn1Y)

**HireSense** is an automated AI pipeline that transforms raw candidate resumes into real-time market opportunities and tailored ATS analytics. 

By combining n8n workflow orchestration with Google Gemini models, HireSense ingests candidate profiles via a web interface, extracts structured data, searches live job opportunities, and generates a responsive, custom-styled ATS analytics report dashboard.

## ✨ Key Features

* **📄 Automated Ingestion:** Direct Webform-to-PDF ingestion with automated structural text extraction.
* **🔍 Live Market Discovery:** Dynamic job discovery executing live queries against real-time job listings via custom HTTP endpoints.
* **🧠 Multi-Model AI Pipeline:** Modular LLM pipeline leveraging 3 Google Gemini models for data normalization, skill extraction, and qualitative job matching.
* **📊 Tailored Report Generation:** Automatically renders custom-themed, interactive HTML/CSS dashboards for candidate ATS scoring and analytics.

## 💡 Overview

HireSense bridges the gap between resume analysis and job hunting, replacing manual searching with a hands-free, agentic workflow that matches candidates to real-world job roles instantly.
